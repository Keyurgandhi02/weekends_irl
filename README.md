# 📱 Weekend IRL — Social Media App

A React Native social platform designed to connect people for real-life weekend activities.  
The goal is simple: **less scrolling, more real-world connections.**

---

## 🚀 Overview
Weekend IRL helps users:
- Discover nearby events & activities  
- Connect with like-minded people  
- Plan real-life meetups instead of just online interaction  

Built as a **weekend project**, focused on speed, performance, and clean UI.

---

## ✨ Features
- 👤 User Authentication (Sign up / Login)
- 📍 Discover nearby events & activities
- 📝 Create & join meetups
- 💬 Basic social interaction (posts / chats)
- 🔔 Real-time updates (if using Firebase)
- 📱 Smooth & responsive UI

---

## 🛠 Tech Stack
- React Native
- Expo (if used)
- Firebase (Auth, Firestore, Storage)
- REST APIs
- State Management (Redux / Context / Zustand)
